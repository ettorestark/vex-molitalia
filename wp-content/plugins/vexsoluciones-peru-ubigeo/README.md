## Installation ##

### Manual Installation ###

1. Upload the entire `/vexsoluciones-peru-ubigeo` directory to the `/wp-content/plugins/` directory.
2. Activate Vexsoluciones Peru Ubigeo through the 'Plugins' menu in WordPress.
